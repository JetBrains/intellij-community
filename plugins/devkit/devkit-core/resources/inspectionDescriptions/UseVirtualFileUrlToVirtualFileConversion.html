Reports VFS lookups whose URL comes from a VirtualFileUrl.

Calling VirtualFileManager.findFileByUrl(virtualFileUrl.getUrl()) bypasses the cached association maintained by the workspace model URL API. Use virtualFileUrl.virtualFile in Kotlin or VirtualFileUrls.getVirtualFile(virtualFileUrl) in Java. These APIs reuse a cached VirtualFile when one is available and retain the existing VFS lookup as a fallback.
