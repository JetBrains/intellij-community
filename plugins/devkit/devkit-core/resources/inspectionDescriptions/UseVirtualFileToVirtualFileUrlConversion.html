Reports conversions from VirtualFile to VirtualFileUrl which bypass the caching conversion API.

Calling VirtualFileUrlManager.getOrCreateFromUrl(virtualFile.getUrl()) loses the already available VirtualFile. Use virtualFile.toVirtualFileUrl(virtualFileUrlManager) in Kotlin or VirtualFileUrls.toVirtualFileUrl(virtualFile, virtualFileUrlManager) in Java. These APIs preserve the association so that a later conversion back to VirtualFile can avoid a VFS lookup.
