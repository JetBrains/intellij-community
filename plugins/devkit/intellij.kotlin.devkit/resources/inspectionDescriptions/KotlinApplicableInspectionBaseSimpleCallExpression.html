Reports Kotlin inspections that extend KotlinApplicableInspectionBase.Simple<KtCallExpression, ...> without overriding getApplicableRanges.

Example

Before:


  internal class MyInspection : KotlinApplicableInspectionBase.Simple<KtCallExpression, Unit>() {
     ...
  }

After:


  internal class MyInspection : KotlinApplicableInspectionBase.Simple<KtCallExpression, Unit>() {
     override fun getApplicableRanges(element: KtCallExpression): List {
         return ApplicabilityRanges.calleeExpression(element)
     }
     ...
  }
