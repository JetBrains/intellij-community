Reports non-serializable types in RPC interfaces (annotated with @kotlinx.rpc.annotations.Rpc).

The inspection checks whether return and parameter types of Kotlin interfaces annotated with @kotlinx.rpc.annotations.Rpc are serializable.

It checks deeply and reports issues on the type usage level in an interface, including a detailed message about which property is non-serializable.

Serializable types:

  • annotated with @kotlinx.serialization.Serializable
  • Primitives, String, Flow, and Deferred are considered serializable by default
