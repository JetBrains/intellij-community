Reports null and empty checks can be simplified.

The quick-fix replaces the null and empty check with the isNullOrEmpty call.

Example:


  fun test(list: List<Int>?) {
      if (list == null || list.isEmpty()) {
          println(1)
      } else {
          println(2)
      }
  }

After the quick-fix is applied:


  fun test(list: List<Int>?) {
      if (list.isNullOrEmpty()) {
          println(1)
      } else {
          println(2)
      }
  }
