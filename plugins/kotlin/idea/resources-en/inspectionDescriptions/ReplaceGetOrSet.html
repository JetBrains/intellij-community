Reports explicit calls of get or set functions which can be replaced by an indexing operator [].

Kotlin allows to provide custom implementations for the predefined set of operators on types. To overload an operator one can mark the corresponding function with the operator modifier: 


    operator fun get(index: Int)
    operator fun set(index: Int, value: Int)
Since operators have predefined symbolic representation the functions above correspond to the indexing one.

Example:


   class Test {
      operator fun get(i: Int): Int = 0
   }

   fun test() {
      Test().get(0) // can be replaced with 'Test()[0]'
   }
