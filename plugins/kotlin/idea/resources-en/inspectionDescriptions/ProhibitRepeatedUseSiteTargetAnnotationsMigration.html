Reports repeated annotation on property accessors which are not marked with @Repeatable annotation and SOURCE retention.

As a result, those annotations will be compiled to the bytecode and it will be unclear which one will be loaded via reflection later.

Example:


annotation class Foo(val x: Int)

@get:Foo(10)
val a: String
    @Foo(20) get() = "foo" // annotation repeated twice but not marked as @Repeatable

This inspection only reports if the Kotlin language level of the project or module is 1.4 or higher.
