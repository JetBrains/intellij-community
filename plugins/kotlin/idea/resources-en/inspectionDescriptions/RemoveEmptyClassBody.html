Reports declarations of classes and objects with an empty body.

Use the 'Remove redundant empty class body' quick-fix to clean up the code.

Examples:


  class EmptyA() {} // <== empty body

  class EmptyB {
      companion object {} // <== empty body
  }

  fun emptyC() {
     object {} // <== anonymous object, it's ok (not reported)
  }

After the quick fix is applied:


  class EmptyA()

  class EmptyB {
      companion object
  }

  fun emptyC() {
     object {}
  }
