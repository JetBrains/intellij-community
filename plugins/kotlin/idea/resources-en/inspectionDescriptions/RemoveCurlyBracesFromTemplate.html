Reports usages of curly braces in a string template around a simple identifier.

'Remove curly braces' quick-fix can be used to amend the code automatically.

Examples:


  fun redundant() {
     val x = 4
     val y = "${x}" // <== redundant
  }

  fun correctUsage() {
      val x = "x"
      val y = "${x.length}" // <== Ok
  }

After the quick-fix is applied:


  fun redundant() {
     val x = 4
     val y = "$x"
  }

  fun correctUsage() {
      val x = "x" <== Updated
      val y = "${x.length}"
  }
