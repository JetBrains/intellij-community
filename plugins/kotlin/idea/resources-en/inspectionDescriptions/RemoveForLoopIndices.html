Reports for loops iterating over a collection using withIndex() function and not using index variable.

Paired "Remove indices in 'for' loop" quick-fix allows to amend highlighted code snippet.

Examples:


fun foo(bar: List<String>) {
  for ((index : Int, value: String) in bar.withIndex()) { // <== 'index' is unused

  }
}

After quick-fix:


fun foo(bar: List<String>) {
   for (value: String in bar) { // <== '.withIndex()' and 'index' are removed

   }
}
