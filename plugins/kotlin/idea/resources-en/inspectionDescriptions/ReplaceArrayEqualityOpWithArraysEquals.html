Reports usages of == or != operator for arrays that should be replaced with contentEquals.

The problem is that both operators compare array references but not the content. The "Replace with 'contentEquals'" quick-fix allows you to amend the code.

Examples:


   fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // can be replaced with 'a.contentEquals(b)'
   }
