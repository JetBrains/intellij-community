Reports usages of == or != operator for arrays that should be replaced with contentEquals.

The problem is that both operators compare array references but not the content. The "Replace with 'contentEquals'" quick-fix allows amending the code.

Examples:


  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // references comparison
  }

After the quick-fix is applied:


  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
