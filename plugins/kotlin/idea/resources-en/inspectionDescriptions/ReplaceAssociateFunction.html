Reports associate() and associateTo() calls that can be replaced with associateBy() or associateWith().

Both functions accept transformer function applied to elements of the given sequence or collection (as a receiver). Provided pairs are used to build resulting Map.
Given the transformer refers to 'it', associate[To]() call can be replaced with more performant associateBy() or associateWith().
The inspection with paired with 'Replace with...' quick-fix allowing to amend the code.

Examples:


   fun getKey(i: Int) = 1L

   fun test() {
      arrayOf(1).associate { getKey(it) to it }  // can be replaced with 'associateBy { getKey(it) }'
      listOf(1).associate { it to getValue(it) } // can be replaced with 'listOf(1).associateWith { getValue(it) }'
   }
