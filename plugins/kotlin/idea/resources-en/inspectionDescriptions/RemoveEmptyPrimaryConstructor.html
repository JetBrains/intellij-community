Reports an empty primary constructor when it would be implicitly available anyway.

Primary constructor can be safely omitted when it does not have any annotations or visibility modifiers. The inspection is paired with 'Remove empty primary constructor' quick-fix.

Examples:


    class MyClassA constructor() //  redundant, can be replaced with 'class MyClassA'

    annotation class MyAnnotation
    class MyClassB @MyAnnotation constructor() //  required because of annotation

    class MyClassC private constructor() // required because of visibility modifier
