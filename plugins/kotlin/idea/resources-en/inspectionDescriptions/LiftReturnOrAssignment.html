Reports if, when, and try statements that can be converted to expressions by lifting a return or an assignment out.

Example:


fun foo(arg: Boolean): String {
    when (arg) {
        true -> return "Truth"
        false -> return "Falsehood"
    }
}
After the quick-fix is applied: 

fun foo(arg: Boolean): String {
    return when (arg) {
        true -> "Truth"
        false -> "Falsehood"
    }
}
