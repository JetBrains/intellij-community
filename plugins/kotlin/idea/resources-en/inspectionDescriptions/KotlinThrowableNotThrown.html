Reports instantiations of Throwable or a subclass, where the created Throwable is never actually thrown.

Example:


fun check(condition: Boolean) {
    if (!condition) /* throw is missing here */ IllegalArgumentException("condition is not met");
}
These are simple mistakes to make which are hard to catch in tests.

Also reports method calls which return instances of Throwable or a subclass, where the result of the method call is not thrown.

Example:


fun createError() = RuntimeException()

fun foo() {
    /* throw is missing here */ createError()
}
