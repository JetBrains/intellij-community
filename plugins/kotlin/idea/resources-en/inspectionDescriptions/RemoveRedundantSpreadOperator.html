Reports the use of a redundant spread operator for a family of arrayOf function calls.

The inspection is paired with 'Remove redundant spread operator' quick-fix to amend the highlighted code snippet.

Examples:


fun foo(vararg xs: String) { }

fun bar(ys: Array<String>) {
   foo(*arrayOf("abc"))       // for the both calls of 'foo' array creation
   foo(*arrayOf(*ys, "zzz"))  // and its subsequent "spreading" is redundant
}

After the quick-fix is applied:


fun foo(vararg xs: String) { }

fun bar(ys: Array<String>) {
   foo("abc")
   foo(*ys, "zzz")
}
