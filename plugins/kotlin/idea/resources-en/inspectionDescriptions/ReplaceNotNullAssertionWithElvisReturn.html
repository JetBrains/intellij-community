Reports not-null assertion (!!) calls that can be replaced with elvis and return (?: return).

A not-null assertion can lead to NPE, which is not always expected. Avoiding the use of !! is good practice.

The quick-fix replaces the not-null assertion with return or return null.

Example:


  fun test(number: Int?) {
      val x = number!!
  }

After the quick-fix is applied:


  fun test(number: Int?) {
      val x = number ?: return
  }
