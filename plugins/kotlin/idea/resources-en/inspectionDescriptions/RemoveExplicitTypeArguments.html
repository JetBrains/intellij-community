Reports function calls with type arguments that can be automatically inferred and safely omitted.

Inspection is paired with 'Remove explicit type arguments' quick-fix allowing to amend the code.

Examples:


   // 'String' type can be inferred here
   fun foo(): MutableList<String> = mutableListOf<String>() <== 'mutableListOf()' is suggested as a replacement

   // Here 'String' cannot be inferred, type argument is required.
   fun bar() = mutableListOf<String>()
