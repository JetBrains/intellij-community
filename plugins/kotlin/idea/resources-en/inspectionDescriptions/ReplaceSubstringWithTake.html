Reports calls like s.substring(0, x) that can be replaced with s.take(x).

Using take() makes your code simpler.

The quick-fix replaces the substring call with take().

Example:


  fun foo(s: String) {
      s.substring(0, 10)
  }

After the quick-fix is applied:


  fun foo(s: String) {
      s.take(10)
  }
