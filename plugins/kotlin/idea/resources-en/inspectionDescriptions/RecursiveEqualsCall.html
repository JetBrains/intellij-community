Reports recursive equals calls.

== can be mistakenly interpreted as reference comparison by Java developers. So the inspection reports such cases and suggests to replace== with ===.

Example:


class X {
    override fun equals(other: Any?): Boolean {
        if (this == other) return true
        return false
    }
}

After the quick-fix is applied:


class X {
    override fun equals(other: Any?): Boolean {
        if (this === other) return true
        return false
    }
}
