Reports calls to toString() in string templates that can be safely removed.

"Remove 'toString()' call" quick-fix can be used to amend the code.

Examples:


   fun foo(a: Int, b: Int) = a + b

   fun test(): String {
      return "Foo: ${foo(0, 4).toString()}" // can be replaced with 'return "Foo: ${foo(0, 4)}"'
   }
