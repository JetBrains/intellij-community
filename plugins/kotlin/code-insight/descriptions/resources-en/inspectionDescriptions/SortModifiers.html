Reports modifiers that do not follow the order recommended by the style guide. If annotations and/or context parameters are present, proves that modifiers follow them and that annotations precede context parameters.

Sort modifiers quick-fix can be used to amend the code automatically.

Examples:


  private inline fun correctOrder(f: () -> Unit) {} // <== Ok

  infix private fun Int.wrongOrder(expr: Int) {} // <== wrong order, quick-fix amends the modifiers to "private infix"


    @Deprecated("message")
    context(str: String)
    private fun doThis() { } // <== Ok


    context(str: String)
    @Deprecated("message")
    fun doThis() { } // <== wrong order, quick-fix moves the annotation to the top
