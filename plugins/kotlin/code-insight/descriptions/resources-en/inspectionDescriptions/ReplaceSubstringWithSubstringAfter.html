Reports calls like s.substring(s.indexOf(x)) that can be replaced with s.substringAfter(x).

Using s.substringAfter(x) makes your code simpler.

The quick-fix replaces the substring call with substringAfter.

Example:


  fun foo(s: String) {
      s.substring(s.indexOf('x'))
  }

After the quick-fix is applied:


  fun foo(s: String) {
      s.substringAfter('x')
  }
