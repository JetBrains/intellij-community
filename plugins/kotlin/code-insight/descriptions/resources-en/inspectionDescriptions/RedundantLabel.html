Reports redundant labels that cannot be referenced from 'break', 'continue' or 'return' statements.

Such labels can be safely removed.

Example:


fun foo(c: Boolean) {
    b@ while (true) a@ {
        while (true) {
            break@b
        }
    }
}

After the quick fix is applied:


fun foo(c: Boolean) {
    b@ while (true) {
        while (true) {
            break@b
        }
    }
}
