Reports if expressions that check whether a list, array, or string is not empty and then return the first element or null.

Using firstOrNull() makes the code shorter and more idiomatic.

The quick-fix replaces the whole if expression with firstOrNull().

Example:


  fun firstChild(children: List<String>): String? {
      return if (children.isNotEmpty()) {
          children[0]
      } else {
          null
      }
  }

After the quick-fix is applied:


  fun firstChild(children: List<String>): String? {
      return children.firstOrNull()
  }
