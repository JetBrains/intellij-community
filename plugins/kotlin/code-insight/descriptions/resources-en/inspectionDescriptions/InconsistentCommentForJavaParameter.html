Reports inconsistent parameter name for java method specified in a comment block.

Examples:


  // Java
  public class JavaService {
      public void invoke(String command) {}
  }


  // Kotlin
  fun main() {
      JavaService().invoke(/* name = */ "fix")
  }

The quick fix corrects parameter name in a comment block:


  fun main() {
      JavaService().invoke(/* command = */ "fix")
  }
