Reports calls to collection factory functions that can be replaced with collection literal syntax.

When the -Xcollection-literals compiler flag is enabled, collection creation functions such as listOf(), setOf(), arrayOf(), and similar functions can be expressed using the more concise collection literal syntax [...].

Example:


val list = listOf("a", "b", "c")

After applying the fix:


val list: List<String> = ["a", "b", "c"]

This inspection requires the -Xcollection-literals compiler flag to be enabled.
