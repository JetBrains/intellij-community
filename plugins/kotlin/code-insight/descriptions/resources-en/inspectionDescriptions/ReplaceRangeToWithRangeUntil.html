Reports calls to rangeTo or the .. operator instead of calls to ..<.

Using corresponding functions makes your code simpler.

The quick-fix replaces rangeTo or the .. call with ..<.

Example:


  fun foo(a: Int) {
      for (i in 0..a - 1) {

      }
  }

After the quick-fix is applied:


  fun foo(a: Int) {
      for (i in 0..<a) {

      }
  }
