Reports an implicit context argument that can be converted to an explicit named argument.

Context parameters are resolved implicitly from in-scope context parameters and receivers.

Example:


  interface Logger { fun log(message: String) }

  context(logger: Logger)
  fun logWithTime(message: String) {
      logger.log("${LocalDateTime.now()}: $message")
  }

  fun saveUser(user: User, logger: Logger) {
      context(logger) {
          logWithTime("saving user ${user.id}")
      }
  }

The quick-fix converts the implicit context argument to explicit:


  fun saveUser(user: User, logger: Logger) {
        logWithTime("saving user ${user.id}", logger = logger)
  }
