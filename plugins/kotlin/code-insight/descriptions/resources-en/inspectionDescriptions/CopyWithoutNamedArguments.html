Reports calls to a data class' copy() method without named arguments.

As all arguments of the copy() function are optional, it might be hard to understand what properties are modified. Providing parameter names explicitly makes code easy to understand without navigating to the data class declaration.

Example:


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy("John")
  }

The quick-fix provides parameter names to all copy() arguments:


  data class User(val name: String, val age: Int)

  fun copyUser(user: User): User {
      return user.copy(name = "John")
  }
