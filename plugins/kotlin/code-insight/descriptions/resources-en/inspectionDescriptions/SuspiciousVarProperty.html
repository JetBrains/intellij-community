Reports var properties with default setter and getter that do not reference backing field.

Such properties do not affect calling its setter; therefore, it will be clearer to change such property to val and delete the initializer.

Change to val and delete initializer quick-fix can be used to amend the code automatically.

Example:


  // This property always returns '1' and it doesn't important that the property is a 'var'
  var foo: Int = 0
      get() = 1
