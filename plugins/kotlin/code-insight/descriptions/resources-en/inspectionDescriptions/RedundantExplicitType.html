Reports local variables' explicitly given types which are obvious and thus redundant, like val f: Foo = Foo().

Example:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t: Boolean = true
      val p: Point = Point(1, 2)
      val i: Int = 42
  }

After the quick-fix is applied:


  class Point(val x: Int, val y: Int)

  fun foo() {
      val t = true
      val p = Point(1, 2)
      val i = 42
  }
