Reports collection concatenations that can be converted to a collection builder call.

The quick-fix replaces chained + and - collection operations with buildList or buildSet.

Example:


fun test(): List<Int> {
    return listOf(1, 2) + listOf(3, 4)
}

After the quick-fix is applied:


fun test(): List<Int> {
    return buildList {
        addAll(listOf(1, 2))
        addAll(listOf(3, 4))
    }
}
