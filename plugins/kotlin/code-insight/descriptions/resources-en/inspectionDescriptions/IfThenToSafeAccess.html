Reports if-then expressions that can be folded into safe-access (?.) expressions.

Example:


  fun bar(x: String) = ""

  fun foo(a: String?) {
     if (a != null) bar(a) else null
  }

The quick fix converts the if-then expression into a safe-access (?.) expression:


  fun bar(x: String) = ""

  fun foo(a: String?) {
     a?.let { bar(it) }
  }
