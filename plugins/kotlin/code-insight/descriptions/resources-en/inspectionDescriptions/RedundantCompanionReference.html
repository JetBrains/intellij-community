Reports redundant Companion reference.

Example:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.Companion.create()
  }

After the quick-fix is applied:


  class A {
      companion object {
          fun create() = A()
      }
  }
  fun test() {
      val s = A.create()
  }
