Reports usages of == or != operator for arrays that should be replaced with contentEquals().

The == and !=operators compare array references instead of their content.

Examples:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a == b) // references comparison
  }

After the quick-fix is applied:

  fun test() {
      val a = arrayOf(1, 2, 3)
      val b = arrayOf(1, 2, 3)
      println(a.contentEquals(b))
  }
