Reports sealed classes that have no properties and can be converted to sealed interfaces.

Sealed interfaces are often preferred over sealed classes when the type hierarchy doesn't need to share state, as they allow implementing classes to extend other classes.

Example:


sealed class Result<out T : Any>
class Success<T : Any>(val result: T) : Result<T>()
class Error(val throwable: Throwable) : Result<Nothing>()

After the quick fix is applied:


sealed interface Result<out T : Any>
class Success<T : Any>(val result: T) : Result<T>
class Error(val throwable: Throwable) : Result<Nothing>
