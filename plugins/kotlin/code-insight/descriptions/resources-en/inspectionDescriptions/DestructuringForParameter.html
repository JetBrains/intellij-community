Enforces using Kotlin idiomatic destructured declarations in for loops when only components of class are used.

Example:


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
    for (item in list) {
      println("${item.second} ${item.third}")
    }
  }

The quick-fix destructures the declaration and introduces new variables with names from the corresponding class:


  data class My(val first: String, val second: Int, val third: Boolean)

  fun foo(list: List<My>) {
    for ((_, second, third) in list) {
      println("$second $third")
    }
  }
