Reports with function calls whose receiver is used only as a context argument inside the lambda.

When the receiver passed to with is never accessed as an implicit receiver (for example, via member calls or this) but only propagated as a context argument to functions declaring context parameters, the with call can be replaced with a context call. This makes the intent explicit and better reflects how the value is actually consumed.

The quick-fix replaces the with callee with context.

Example: 


interface UserService {
    fun log(message: String)
}

context(service: UserService)
fun logStart() {
    service.log("start")
}

fun run(service: UserService) {
    with(service) {
        logStart()
    }
}

After the quick-fix is applied:


fun run(service: UserService) {
    context(service) {
        logStart()
    }
}

This inspection requires the ContextParameters language feature to be enabled.
