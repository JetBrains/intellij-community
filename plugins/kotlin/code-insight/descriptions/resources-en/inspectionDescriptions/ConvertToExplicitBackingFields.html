Detects properties with backing fields and suggests a refactoring introducing backing fields explicitly using a field keyword.

See https://github.com/Kotlin/KEEP/blob/main/proposals/KEEP-0430-explicit-backing-fields.md for more information.

Example:


  private val _items = mutableListOf()
  val items: List get() = _items

After the quick-fix is applied:


  val items: List
    field = mutableListOf()
