Reports sealed interfaces that have no properties and can be converted to sealed classes.

Sealed classes may be preferred over sealed interfaces when you want to ensure a single inheritance hierarchy or plan to add state (properties) in the future.

Example:


sealed interface Result<out T : Any>
class Success<T : Any>(val result: T) : Result<T>
class Error(val throwable: Throwable) : Result<Nothing>

After the quick fix is applied:


sealed class Result<out T : Any>
class Success<T : Any>(val result: T) : Result<T>()
class Error(val throwable: Throwable) : Result<Nothing>()
