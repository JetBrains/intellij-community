Detects uses of lazy { ... } without delegation.

Example:


    private val x = lazy { "hello" }

    fun test() {
        printlin(x.value)
    }
A quick-fix is suggested if a lazy variable is only used by calling direct access to its value: 

  private val x by lazy { "hello" }

  fun test() {
        println(x)
  }

Constructions with explicit lazy type like 


    private val x: Lazy<String> = lazy { "hello" }
are not reported, as they are assumed to be used intentionally.
