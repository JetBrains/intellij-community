Reports an explicit context argument that can be converted to an implicit one by wrapping the call in a context(...) { } block.

Context parameters are implicit: their values are resolved from in-scope context parameters and implicit receivers. Wrapping the call in a context block adds the value to the context, making it available for implicit resolution.

Example:


  interface Logger { fun log(message: String) }

  context(logger: Logger)
  fun logWithTime(message: String) {
      logger.log("${LocalDateTime.now()}: $message")
  }

  fun saveUser(user: User, logger: Logger) {
      logWithTime("saving user ${user.id}", logger = logger)
  }

The quick-fix wraps the call in a context block:


  fun saveUser(user: User, logger: Logger) {
      context(logger) {
          logWithTime("saving user ${user.id}")
      }
  }
