Detects registration of tasks without assigning them a group or description. Suggests a quick fix to add a template for group and description fields.

Example:


tasks.register("generateDocs") { // reports method 'register'
    // Build logic to generate documentation
}

After the quick fix is applied:


tasks.register("generateDocs") {
    group = "<caret>"
    description = ""
    // Build logic to generate documentation
}

Best practice described here.
