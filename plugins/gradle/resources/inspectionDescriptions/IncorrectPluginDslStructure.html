Detects disallowed statements before 'plugins {}' block.

Due to the limitations of Gradle Plugin DSL, only a restricted set of Groovy statements is available before 'plugins {}' block. The only options are 'buildscript {}', 'pluginManagement {}' and other 'plugins {}'.
See Gradle documentation

Example:


import foo.bar.Baz
plugins {} // reports 'plugins'
plugins {
  foo() // reports 'foo'
  id 'java'
}
