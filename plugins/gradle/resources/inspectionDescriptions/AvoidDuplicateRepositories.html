Detects duplicate repository declarations in Gradle scripts.

Offers an intention to navigate to duplicates.

Inspection reporting example:


repositories {
    // the following two repositories will be reported
    maven { url = uri("https://repo1.maven.org/maven2/") }
    maven {
        url = uri("https://repo1.maven.org/maven2/")
        // some comment
    }
    maven { url = uri("https://some.other.repo/") } // will not be reported
}
