This module is needed only for `IDEA Frontend (dev build, Light Mode, with LSP UI)` run configuration.

It provides [product-modules.xml](resources/META-INF/intellij.lsp.ui.idea.product.modules/product-modules.xml) that bundles the `com.intellij.lsp.ui` plugin into the JetBrains Client dev build.
It shouldn't be used in production.
