Reports suspicious getter or setter methods. A getter or setter is suspicious if it accesses a different field than would be expected by its name.

In many cases it may be a copy-paste error.

Example:


  class Point {
    private int x;
    private int y;

    public void setX(int x) { // Warning: setter 'setX()' assigns field 'y'
      this.y = x;
    }

    public int getY() { // Warning: getter 'getY()' returns field 'x'
      return x;
    }
  }
