Reports any declaration of an equal() method with just one argument. Normally, this is a typo of equals().

The quick-fix is provided to rename the method to 'equals'

Example:

  
    class Main {
      public boolean equal(Object obj) {
        return true;
      }
    }

After the quick-fix is applied:

  
    class Main {
      public boolean equals(Object obj) {
        return true;
      }
    }
