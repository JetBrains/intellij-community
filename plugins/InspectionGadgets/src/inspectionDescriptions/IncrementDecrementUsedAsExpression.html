Reports increment or decrement expressions that are nested inside other expressions. Such expressions may be confusing and violate the general design principle, which states that any construct should do precisely one thing.

The quick-fix extracts the increment or decrement operation to a separate expression statement.

Example:


  int i = 10;
  while (i-- > 0) {
    System.out.println(i);
  }

After the quick-fix is applied:


  int i = 10;
  while (i > 0) {
    i--;
    System.out.println(i);
  }
  i--;
