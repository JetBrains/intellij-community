Reports switch statements with too few case labels, and suggests rewriting it as if and else ifstatements.

Example (minimum branches == 3):

switch (expression) {
  case "foo" -> foo();
  case "bar" -> bar();
}

After the quick-fix is applied the result looks like:

if ("foo".equals(expression)) {
  foo();
}
else if ("bar".equals(expression)) {
  bar();
}

Use the field below to specify the minimum number of case labels expected.
