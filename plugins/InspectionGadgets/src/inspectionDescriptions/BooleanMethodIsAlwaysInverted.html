Reports methods with a boolean return type, which are only used in a negated context. Due to performance reasons some methods might not be reported during in-editor highlighting.

Example:

  class C {
    boolean alwaysInverted() {
      return true;
    }

    void f() {
      if (!alwaysInverted()) {
        return;
      }
    }
    boolean member = !alwaysInverted();
  }

After the quick-fix is applied (the function name can be changed), the result looks like this:

  class C {
    boolean notInvertedAnymore() {
      return false;
    }

    void f() {
      if (notInvertedAnymore()) {
        return;
      }
    }
    boolean member = notInvertedAnymore();
  }
