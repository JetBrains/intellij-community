Reports incorrect date time format patterns.

The following errors are reported:

  • Unsupported patterns, like "TT"
  • Using reserved patterns, like "#"
  • Unappropriated padding.
  • Optional brackets without a pair.

Examples:
DateTimeFormatter.ofPattern("[][]]])"
 ("]" doesn't have previous "[") DateTimeFormatter.ofPattern("TT")
 (This pattern contains unsupported token "TT") DateTimeFormatter.ofPattern("{")
 (This pattern contains reserved character "{")

New in 2022.3
