This inspection reports == and != equality operators that compare two objects. For example:
if (object1 == object2) {}

Quick fix replaces '==' with .equals():

object1.equals(object2);

or with safe .equals():

  • java.util.Objects.equals(object1, object2)
    (under Java7+) or
  • object1 == null ? object2 == null : object1.equals(object2)
    (under old JDKs)
