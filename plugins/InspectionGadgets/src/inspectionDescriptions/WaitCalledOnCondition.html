Reports on any call to wait() made on a java.util.concurrent.locks.Condition object. This is probably a programming error, and some variant of the await() method was intended instead.

Example:

void acquire(Condition released) throws InterruptedException {
  while (acquired) {
    released.wait();
  }
}

Good code would look like this: 

void acquire(Condition released) throws InterruptedException {
  while (acquired) {
    released.await();
  }
}
