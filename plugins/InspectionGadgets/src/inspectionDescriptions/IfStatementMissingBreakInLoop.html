Reports cases when loop contains if statement and this if statement can end with break.

For instance, consider the following code: 


  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
    }
  }
In this case iterations have no effect after condition is met and you may skip them adding a break. 

  boolean found = false;
  for (int i = 0; i < arr.length; i++) {
    if (Objects.equals(value, arr[i])) {
      found = true;
      break;
    }
  }
