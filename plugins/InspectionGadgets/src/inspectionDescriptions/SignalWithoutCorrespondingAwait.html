Reports on any call to Condition.signal() or Condition.signalAll() for which no call to a corresponding Condition.await() can be found. Only calls which target fields of the current class are reported by this inspection.

Example:


  class Queue {
    private final Condition isEmpty = ...;

    void add(Object elem) {
      // ...
      isEmpty.signal(); // warning: Call to 'signal()' without corresponding 'await()'
      // ...
    }

    void remove(Object elem) throws InterruptedException {
      // ...
      //      isEmpty.await();
      // ...
    }
  }
