Reports usages of StringBuffer, StringBuilder, or StringJoiner which can be replaced with a single String concatenation.

Using String concatenation makes the code shorter and simpler.

This inspection only reports when the suggested replacement does not result in significant performance drawback on modern JVMs. In many cases, String concatenation may perform better.

Example:

  StringBuilder result = new StringBuilder();
  result.append("i = ");
  result.append(i);
  result.append(";");
  return result.toString();

After the quick-fix is applied:

  String result = "i = " + i + ";";
  return result;
