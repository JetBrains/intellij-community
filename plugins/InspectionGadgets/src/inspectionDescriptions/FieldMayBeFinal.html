Reports any fields that can be safely made final.

Using the final variable makes code less error-prone. It may also be helpful for compiler optimisations that make your program run faster.

A field may be safely made final only if it has the private access modifier (or it is defined in a local / anonymous class) and:

  • It is static and initialized either in its declaration or in one static class initializer.
  • It is non-static and initialized either in its declaration or in one non-static class initializer or in all constructors.
