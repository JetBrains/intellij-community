Reports any references to local non-Serializable variables outside serializablelambda's, local and anonymous classes.

When a local variable is referenced from an anonymous class, it's value is stored in an implicit field in the anonymous class. The same happens for local classes and lambda's. If that variable is of a non-Serializable type, serialization will fail.

Example:


  interface A extends Serializable {
      abstract void foo();
  }
  class B {}
  class C {
      void foo() {
          B b = new B();
          A a = new A() {
              @Override
              public void foo() {
                  System.out.println(b); // warning
              }
          };
      }
  }
