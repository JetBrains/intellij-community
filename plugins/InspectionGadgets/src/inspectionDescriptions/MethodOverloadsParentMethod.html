Reports instance methods having the same name and the same number of parameters with the same or compatible types as a method in a superclass. In this case, the child method overloads the parent method, instead of overriding it. If it is unintended, it may result in latent bugs.

Example:


  public class Foo {
    void foo(int x) {}
  }

  public class Bar extends Foo {
    void foo(Number x) {} // Method 'foo()' overloads a compatible method of a superclass,
                          // when overriding might have been intended
  }
