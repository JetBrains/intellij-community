Reports enhanced for statements.

Example:


  for (int x: Arrays.asList(1, 2, 3)) {
    System.out.println(x);
  }

After the quick-fix is applied: 


  for (Iterator<Integer> iterator = Arrays.asList(1, 2, 3).iterator(); iterator.hasNext(); ) {
    final int x = iterator.next();
    System.out.println(x);
  }

Enhanced for statement appeared in Java 5. This inspection can help to downgrade for backward compatibility with earlier Java versions.
