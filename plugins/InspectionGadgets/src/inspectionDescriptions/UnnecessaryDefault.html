Reports enum switch statements or expression with default branches which can never be taken, because all possible values are covered by a case branch. These are unnecessary, especially for switch expressions, because they don't compile when all enum constants are not covered by a case branch.

The language level needs to be configured to 14 to report switch expressions.

Quick fix removes default branch.

Example:

  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
      default -> 3;
    };
  }

After the quick-fix is applied the result looks like:

  enum E { A, B }
  int foo(E e) {
    return switch (e) {
      case A -> 1;
      case B -> 2;
    };
  }

Use option to report only unnecessary default branches in switch expressions
