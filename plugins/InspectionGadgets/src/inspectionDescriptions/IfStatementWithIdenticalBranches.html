Reports if statements where common parts can be extracted from the branches. These common parts are independent from the condition and make if statements harder to understand.

Example:

  if (x > 12) {
    doSomethingBefore();
    doSomethingDifferent1();
    doSomethingAfter();
  } else {
    doSomethingBefore();
    doSomethingDifferent2();
    doSomethingAfter();
  }

After the quick-fix is applied, the result looks like this:

  doSomethingBefore();
  if (x > 12) {
    doSomethingDifferent1();
  } else {
    doSomethingDifferent2();
  }
  doSomethingAfter();
