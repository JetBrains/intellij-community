Reports instantiations of java.util.Map objects whose key types are enumerated classes. Such java.util.Map objects can be replaced with java.util.EnumMap objects.

java.util.EnumMap implementations can be much more efficient because the underlying data structure is a simple array.

Example:


  Map<MyEnum, String> myEnums = new HashMap<>();

After the quick-fix is applied:


  Map<MyEnum, String> myEnums = new EnumMap<>(MyEnum.class);
