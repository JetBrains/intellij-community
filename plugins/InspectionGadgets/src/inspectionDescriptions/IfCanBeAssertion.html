Reports if statements which only throw a java.lang.Throwable from the then branch, and don't have an else branch. Also reports Guava's Preconditions.checkNotNull(). These can be replaced with an assert statement, or Objects.requireNonNull() call.

Example: 


  if (x == 2) throw new RuntimeException("fail");
  if (y == null) throw new AssertionError();
  Preconditions.checkNotNull(z, "z");

After the quick fix is applied the result looks like:


  assert x != 2 : "fail";
  Objects.requireNonNull(y);
  Objects.requireNonNull(z, "z");
