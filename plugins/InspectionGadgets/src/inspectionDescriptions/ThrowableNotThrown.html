Reports instantiations of Throwable or a subclass, where the created Throwable is never actually thrown. These are simple mistakes to make which are hard to catch in tests.

Also reports method calls which return instances of Throwable or a subclass, where the result of the method call is not thrown. Calls to methods annotated with Error Prone's or AssertJ's @CanIgnoreReturnValue annotation will not be reported.

Example:

    void leftBehind() throws Throwable {
        RuntimeException e = new RuntimeException("throw me");
        // ...
    }
