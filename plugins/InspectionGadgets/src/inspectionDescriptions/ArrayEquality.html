Reports any use of == or != to test for array equality, rather than the java.util.Arrays.equals() method testing for content equality.

Apply the quick-fix to replace == by java.util.Arrays.equals().

Example:


  void foo(Object[] x, Object[] y) {
    boolean comparison = x == y;
  }

After the quick-fix is applied:


  void foo(Object[] x, Object[] y) {
    boolean comparison = Arrays.equals(x, y);
  }
