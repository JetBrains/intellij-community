Reports methods with three or more negation operators ! or !=. Such methods may be unnecessarily confusing.

For example:

  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (!flag && !flag2) {
      if (a != b) {
        doOther();
      }
    }
  }

This method will be easier to understand without negations:

  void doSmth(int a, int b, boolean flag1, boolean flag2) {
    if (flag1 || flag2 || a == b) return;
    doOther();
  }

Use the second option below to disable this inspection within assert statements.
