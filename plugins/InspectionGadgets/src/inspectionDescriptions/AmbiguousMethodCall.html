Reports calls to any super method from inner or anonymous class if a method with the identical signature exists in the surrounding class. In this case it's easy to miss the super method and suggest that the call is referenced to method in the surrounding class. To clarify the intent it is recommended to add an explicit super qualifier to the method call.

Example:

  class Parent {
    void ambiguous(){}
  }

  class Example {
    void ambiguous(){}

    class Inner extends Parent {
      void example(){
        ambiguous(); //warning
      }
    }
  }

The quick-fix adds an explicit super qualifier to the method call:

  super.ambiguous();
