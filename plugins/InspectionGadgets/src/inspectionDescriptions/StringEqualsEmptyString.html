Reports cases when .equals() is called to compare a String with an empty string. In this case, using .isEmpty() is better as as it shows you exactly what you're checking.

Some checks, for example, the "".equals(str) check, return false when str is null. It means that an explicit null-check could be inserted when you are replacing .equals() with isEmpty(). Use a checkbox below to suppress the warning when a string value could be nullable.
