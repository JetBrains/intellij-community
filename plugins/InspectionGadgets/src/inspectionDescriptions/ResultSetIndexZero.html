Reports any attempts to access column 0 of a java.sql.ResultSet or java.sql.PreparedStatement. For historical reasons columns of java.sql.ResultSets and java.sql.PreparedStatements are numbered beginning with 1, rather than 0, and accessing column 0 is a common error in JDBC programming.

Example:


  String getName(ResultSet rs) {
    return rs.getString(0);
  }
