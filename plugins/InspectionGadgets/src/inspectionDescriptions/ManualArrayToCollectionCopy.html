Reports copying array content into collection in a loop instead of using Collection.addAll(Arrays.asList()) or Collections.addAll() methods. Replacing loops make your code shorter and potentially faster depending on collection implementation

Note that only loops that do not have any additional statements inside are reported.

Example:

  void addAll(List<String> list, String[] arr) {
    for (int i = 0; i < arr.length; i++) {
      String s = arr[i];
      list.add(s);
    }
  }

After the quick-fix is applied the result looks like:

  void addAll(List<String> list, String[] arr) {
    Collections.addAll(list, arr);
  }
