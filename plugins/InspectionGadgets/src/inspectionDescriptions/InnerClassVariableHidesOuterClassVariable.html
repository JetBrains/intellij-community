Reports inner class variables being named identically to member variables of a containing class. Such a variable name may be confusing, and can be renamed using the quick-fix

Example:


  class Chief {
    private String name;

    class Paramount {
      private String name;
    }
  }

After the quick-fix is applied:


  class Chief {
    private String name;

    class Paramount {
      private String identifier;
    }
  }

Use the checkbox below the indicate whether this inspection should report all name clashes, or only clashes with fields which are visible from the inner class.
