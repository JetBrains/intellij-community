Reports break statements with unnecessary labels. Such labels do not change the control flow but make the code more complicated to follow.

Example:

    label: 
    for(int i = 0; i < 10; i++) {
      if (shouldBreak()) break label;
      //doSmth
    }

After the quickfix is applied the result looks like:

    label: 
    for(int i = 0; i < 10; i++) {
      if (shouldBreak()) break;
      //doSmth
    }
