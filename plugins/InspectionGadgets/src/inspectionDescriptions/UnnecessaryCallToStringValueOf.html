Reports calls to static methods like String.valueOf() or Integer.toString() when they are used in a string concatenation or as an argument of a library method where the explicit string conversion is not needed.

Example:

  System.out.println("Number: " + Integer.toString(count));

After the quick-fix applied the result looks like:

  System.out.println("Number: " + count);

Library methods where explicit string conversion considered as not needed:

  • classes java.io.PrintWriter, java.io.PrintStream
    • print(), println()
  • classes java.lang.StringBuilder, java.lang.StringBuffer
    • append()
  • class org.slf4j.Logger
    • trace(), debug(), info(), warn(), error()
