Reports methods which body is contained in a single synchronized statement. The lock expression for this synchronized statement must be equal to this for instance methods or ClassName.class for static methods. In such cases the synchronized statements may be replaced by their contents and the containing method marked as synchronized.

Example: 

  public int generateInt(int x) {
    synchronized (this) {
      return 1;
    }
  }

After applying the quick-fix the code looks as follows:

  public synchronized int generateInt(int x) {
    return 1;
  }
