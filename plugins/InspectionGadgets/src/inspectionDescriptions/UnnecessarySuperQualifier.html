Reports any unnecessary uses of the super qualifier in method calls and fields references. A super qualifier is unnecessary when the field or method of the super class is not overridden in the calling class.

Use the checkbox below to ignore qualifiers, which help to distinguish super members access from the same named members of the outer class. See also

  • Java | Visibility issues | Access of inherited field looks like access of element in surrounding code
  • Java | Visibility issues | Call to inherited method looks like call to local method
