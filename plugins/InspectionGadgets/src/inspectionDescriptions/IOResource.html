Reports any I/O resource which is not safely closed. I/O resources checked by this inspection include java.io.InputStream, java.io.OutputStream, java.io.Reader, java.io.Writer, java.util.zip.ZipFile, java.io.Closeable and java.io.RandomAccessFile. I/O resources which are wrapped by other I/O resources are not reported, as the wrapped resource will be closed by the wrapping resource.

By default, inspection assumes that the resources can be closed by any method with a close or cleanup in the name.

Example:

  void save() throws IOException {
    FileWriter writer = new FileWriter("filename.txt"); //warning
    writer.write("sample");
  }
