Reports unnecessarily escaped characters in String and optionally char literals.

The escaped tab character \t is not reported, because otherwise it will be invisible.

Examples:


  String s = "\'Scare\' quotes";
  String t = """
    All you need is\n\tLove\n""";

After the quick-fix is applied:


  String s = "'Scare' quotes";
  String t = """
    All you need is
    \tLove
  """;

New in 2019.3
