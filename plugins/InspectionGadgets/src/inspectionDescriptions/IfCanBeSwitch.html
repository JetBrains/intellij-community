Reports any if statements that can be replaced with a switch statement. The result of transformation is usually shorter and clearer.

Example:

  void test(String str) {
    if (str.equals("1")) {
      System.out.println(1);
    } else if (str.equals("2")) {
      System.out.println(2);
    } else if (str.equals("3")) {
      System.out.println(3);
    } else {
      System.out.println(4);
    }
  }

After quick-fix is applied, the result will be:

  void test(String str) {
    switch (str) {
      case "1" -> System.out.println(1);
      case "2" -> System.out.println(2);
      case "3" -> System.out.println(3);
      default -> System.out.println(4);
    }
  }

This inspection only reports if the language level of the project or module is 7 or higher.
