Reports methods which are highly coupled, i.e. that reference too many other classes. Methods with too high coupling can be very fragile, and should probably be broken up.

Note: every referenced class is counted only once no matter how many times it was referenced.

Use the field below to specify the maximum coupling a method is allowed to have.

Use the first checkbox below to count references to classes from javaor javax packages

Use the second checkbox below to count references to third-party library classes
