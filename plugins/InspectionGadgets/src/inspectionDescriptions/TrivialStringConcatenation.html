Reports string concatenations where one of the arguments is the empty string. Such a concatenation is unnecessary and inefficient, particularly when used as an idiom for formatting non-String objects or primitives into String-s.

Apply the quick-fix to simplify the concatenation.

Example:


  void foo(int x, int y) {
    String s = "" + x + " ; " + y;
  }

After the quick-fix is applied:


  void foo(int x, int y) {
    String s = x + " ; " + y;
  }
