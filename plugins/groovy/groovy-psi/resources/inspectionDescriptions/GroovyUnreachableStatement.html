Reports statements that are unreachable. This can occur if the statement is after an infinite loop, return, break, or continue statement.

Example: 


  void foo (int n) {
      if (n < 1) {
          return
          print('This statement is unreachable')
      }
      while (true){
          print ('Hello, world!')
      }
      print('This statement is unreachable too')
  }
