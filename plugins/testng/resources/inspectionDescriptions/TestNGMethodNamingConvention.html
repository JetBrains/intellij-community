Reports TestNG test methods whose names are either too short, too long, or do not follow the specified regular expression pattern.

When this inspection is enabled, the Instance method naming convention inspection ignores TestNG test methods automatically.

Use the fields to specify minimum length, maximum length, and a regular expression expected for TestNG test method names. Specify 0 in order not to check the length of names. Regular expressions are specified in the standard java.util.regex format.
