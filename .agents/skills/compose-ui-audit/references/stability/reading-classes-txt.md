# Reading classes.txt

`classes.txt` lists every class, interface, enum, and typealias in the compiled module and states
whether the Compose compiler considers it stable, unstable, or runtime-stable.

## File structure

```
Stable klass MyPackage.DataClass
  stable val name: String
  stable val count: Int

Unstable class MyPackage.MutableThing
  unstable var name: String
  stable val count: Int

Runtime class MyPackage.ExternalWrapper
  runtime stable val data: ExternalType
```

## Three top-level verdicts

| Verdict | Meaning | Follow-up |
|---------|---------|-----------|
| `Stable` | All properties are stable and immutable. | None needed. |
| `Unstable` | At least one property is mutable or of an unstable type. | Read the per-property lines to find the culprit. |
| `Runtime` | The compiler inserted a `$stable` field; stability is decided at runtime by a bitmask. | Usually behaves as stable. Look at `$stable` in the generated bytecode only if you suspect a bug. |

## Per-property prefixes

| Prefix | Meaning |
|--------|---------|
| `stable` | Immutable, stable type. |
| `unstable` | Mutable (`var`) or of an unstable/unknown type. |
| `runtime stable` | The property type has a `$stable` field; stability is deferred to runtime. |

## Common causes of `Unstable`

1. **A `var` field.** The compiler sees mutability and pessimistically marks the whole class
   unstable, even if the `var` is never written after construction.
2. **A `List<T>`, `Map<K,V>`, `Set<T>` property.** The `kotlin.collections.*` interfaces are
   interfaces; the compiler cannot guarantee every implementation is immutable.
3. **A property of a Java class or third-party type.** Java classes without `@Immutable` or
   `@Stable` are presumed unstable.
4. **A generic property `T` where `T` is unbounded.** The compiler cannot prove `T` is stable.
5. **A nested unstable type.** The outer class is unstable because an inner property is unstable.

## Search patterns

```bash
# Find every unstable class
grep -n "Unstable" classes.txt

# Find the details for a specific class
grep -n -A 10 "Unstable class MyPackage.MyClass" classes.txt
```
