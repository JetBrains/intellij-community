Reports any declarations of classes, fields, or methods marked with @org.jetbrains.annotations.ApiStatus.ScheduledForRemoval annotation that must already be removed. Specified scheduled removal version is compared with the current version that you may set below. Versions are dot-separated strings with possibly "alpha", "beta", "eap" classifiers, like "1.0", "2.3.1", "2018.1", "7.5-snapshot", "3.0-eap". Versions comparison is intuitive: 1.0 < 2.0, 1.0-eap < 1.0, 2.3-snapshot < 2.3 and so on. For detailed comparison logic refer to implementation of VersionComparatorUtil.
