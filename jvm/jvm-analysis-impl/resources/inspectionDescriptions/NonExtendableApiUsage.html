Reports classes, interfaces and methods that extend, implement or override API elements marked with an @ApiStatus.NonExtendable annotation. An @ApiStatus.NonExtendable annotation indicates that the annotated API class, interface or method must not be extended, implemented or overridden. Instances of classes and interfaces marked with this annotation may be cast to an internal implementing class in the library code, leading to ClassCastException if a different implementation is provided by client. New abstract methods may be added to such classes and interfaces breaking compatibility with client's implementations.
