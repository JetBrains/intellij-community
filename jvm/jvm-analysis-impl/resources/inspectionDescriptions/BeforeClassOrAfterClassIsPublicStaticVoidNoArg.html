Reports JUnit @BeforeClass, @AfterClass, @BeforeAll and @AfterAll methods that are not declared public and static, do not return void, or take arguments. Such methods are easy to create inadvertently, but they will not be executed by JUnit tests runners.

Example:


  public class MainTest {
    @BeforeClass
    String beforeClass(int i) { ... }
  }

After the quick-fix is applied:


  class MainTest {
    @BeforeClass
    public static void beforeClass() { ... }
  }
