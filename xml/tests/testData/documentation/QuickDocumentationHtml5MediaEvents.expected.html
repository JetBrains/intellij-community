onstalled
Widely available across major browsers

Chrome 3, Chrome Android 18, Edge 12, Firefox 3.5, Firefox Android 4, Opera 10.5, Safari 3.1, Safari iOS 3

Baseline since 2015

The stalled event is fired when the user agent is trying to fetch media data, but data is unexpectedly not forthcoming. This event is not cancelable and does not bubble.
