Reports violations of the PEP8 naming conventions.

Example:


class mammalia(object):
    extremities = 4

    def feeds(self):
        print("milk")

In this code fragment, IDE offers to rename mammalia to Mammalia. When the quick-fix is applied, the code change to:


class Mammalia(object):
    extremities = 4

    def feeds(self):
        print("milk")
