This intention prefixes a string literal with "f" to make it an f-string literal (for Python versions >= 3.6).

The string must meet the following conditions:
  • is not a docstring
  • not prefixed with "f" or "F"
  • not prefixed with "u" or "U"
  • not prefixed with "b" or "B"
  • contains at least one "{"
  • terminated with the same type of quotes it begins with
