Reports the cases when you rewrite a loop variable with an inner loop: 
    for i in range(5):
      for i in range(20, 25):
          print("Inner", i)
      print("Outer", i)
It also warns you if a variable declared in the with statement is redeclared inside of the statement body: 
    with open("file") as f:
      f.read()
      with open("file") as f:
