Reports invalid definition of a typing.NamedTuple.

Example:

import typing


class FullName(typing.NamedTuple):
    first: str
    last: str = ""
    middle: str

As a fix, place the field with the default value after the fields without default values:

import typing


class FullName(typing.NamedTuple):
    first: str
    middle: str
    last: str = ""
