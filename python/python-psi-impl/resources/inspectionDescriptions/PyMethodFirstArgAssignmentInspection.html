Reports cases when the first parameter, such as self or cls, is reassigned in a method. Because in most cases, there are no objectives in such reassignment, the IDE indicates an error.

Example:


class Account:
    def calc(self, balance):
        if balance == 0:
            self = balance
        return self

As a fix, you might want to check and modify the algorithm to ensure that reassignment is needed. If everything is correct, you can invoke intention actions for this code and opt to ignore the warning.
