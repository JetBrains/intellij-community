Reports a problem when instance attribute definition is outside __init__ method.

Example:


    class Book:
    def __init__(self):
        self.author = 'Mark Twain'

    def release(self):
        self.year = '1889'

When the quick-fix is applied, the code sample changes to: 


    class Book:
    def __init__(self):
        self.year = '1889'
        self.author = 'Mark Twain'

    def release(self):
        pass
