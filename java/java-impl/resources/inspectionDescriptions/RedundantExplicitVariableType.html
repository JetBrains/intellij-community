Reports local variable types that can be inferred from context and replaced with var.

Example:


  void test(InputStream s) {
    try (InputStream in = s) {}
  }

After the fix is applied:


  void test(InputStream s) {
    try (var in = s) {}
  }

Use the Only report when variable initializer is "simple" checkbox to only report variables with a new expression or literal initializer.
