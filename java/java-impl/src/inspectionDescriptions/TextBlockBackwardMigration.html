Reports text blocks which can be replaced with a regular string literal. Example: 

  Object obj = engine.eval("""
                           function hello() {
                             print('"Hello, world"');
                           }

                           hello();
                           """);

can be replaced with


  Object obj = engine.eval("function hello() {\n" +
                           "  print('\"Hello, world\"');\n" +
                           "}\n" +
                           "\n" +
                           "hello();\n");

This inspection works if the language level is 13 Preview.

New in 2019.3
