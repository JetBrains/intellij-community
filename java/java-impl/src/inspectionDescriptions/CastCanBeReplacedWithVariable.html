Reports type cast operations that can be replaced with existing local or pattern variables with the same value.

Example:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(((String) obj).trim());
  }

After the quick-fix is applied:


  void foo(Object obj) {
    String s = (String) obj;
    System.out.println(s.trim());
  }
