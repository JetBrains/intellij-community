Reports copy constructors that don't copy all the fields of the class.

final fields with initializers and transient fields are considered unnecessary to copy.

Example:


  class Point {

    private int x;
    private int y;

    Point(int x, int y) {
      this.x = x;
      this.y = y;
    }

    Point(Point other) {
      // fields x and y are not initialized
    }
  }
