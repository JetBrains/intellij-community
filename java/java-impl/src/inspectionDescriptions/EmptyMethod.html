This inspection reports methods where:

 - method is empty OR
 - all implementations of interface method are empty OR
 - method is empty itself and is overridden only by empty methods

Note that a method containing only the super() call and passing its own parameter is also considered empty.

This inspection is automatically suppressed for methods annotated with special annotations, for example, EJB annotations javax.ejb.Init and javax.ejb.Remove. Additional annotations can be configured below.
