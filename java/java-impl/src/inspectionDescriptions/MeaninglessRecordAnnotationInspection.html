Reports annotations on the record components that have no effect. This may happen in two cases:

  • Annotation has METHOD target but the corresponding accessor is explicitly defined
  • Annotation has PARAMETER target but the canonical constructor is explicitly defined

Example:

@Target(ElementType.METHOD)
@interface A { }
  
// The annotation will not appear in bytecode at all,
// as it should be propagated to the accessor but accessor is explicitly defined 
record R(@A int x) {
  public int x() { return x; }
}

New in 2021.1
