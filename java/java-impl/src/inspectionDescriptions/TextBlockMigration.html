Reports cases when string or concatenation of strings can be replaced with a text block. Example: 

  String html = "\n" +
                "    \n" +
                "        
Hello, world
\n" +
                "    \n" +
                "\n";

can be replaced with


  String html = """
                
                    
                        
Hello, world

                    
                
                """;

This inspection works if the language level is 13 Preview.

New in 2019.3
