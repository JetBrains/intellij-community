Reports when a string or concatenation of strings can be replaced with a text block. Example: 

  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

can be replaced with


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;

This inspection only reports if the configured language level is 14 Preview.

New in 2019.3
