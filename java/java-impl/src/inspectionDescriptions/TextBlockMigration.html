Suggests to replace string literals with text blocks.

Requirements:

  • \n occurs two or more times.
  • Text blocks are not concatenated.

Use the Apply to single string literals option to suggest the fix for single literals containing line breaks.

Example: 


  String html = "<html>\n" +
                "    <body>\n" +
                "        <p>Hello, world</p>\n" +
                "    </body>\n" +
                "</html>\n";

can be replaced with


  String html = """
                <html>
                    <body>
                        <p>Hello, world</p>
                    </body>
                </html>
                """;

This inspection only reports if the configured language level is 15 or higher.

New in 2019.3
