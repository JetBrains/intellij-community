Informs you when bitwise operations can be replaced with the Long.hashCode method. It detects instances of the pattern (int)(x ^ (x >>> 32)) where x is a variable of type long. This improves readability of code.

Example: 


int result = (int)(var ^ (var >>> 32));
After applying the quick-fix: 

int result = Long.hashCode(var);

This inspection only reports if the language level of the project or module is 8 or higher.
