Reports getter or setter methods that access a field that is not expected by its name. For example, when getY() returns the x field. Usually, it might be a copy-paste error.

Example:

class Point {
    private int x;
    private int y;

    public void setX(int x) { // Warning: setter 'setX()' assigns field 'y'
      this.y = x;
    }

    public int getY() { // Warning: getter 'getY()' returns field 'x'
      return x;
    }
  }
