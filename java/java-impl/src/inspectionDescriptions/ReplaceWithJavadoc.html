Reports a regular comment that belongs to a field, method, or class that can be replaced with a Javadoc comment.

Example:


  public class Main {
    /*
    * Hello,
    */
    // World!
    void f() {
    }
  }

After the quick-fix is applied:


  public class Main {
    /**
     * Hello,
     * World!
     */
    void f() {
    }
  }
