Reports classes that are directly or indirectly dependent on too many other classes.

Modifications to any dependency of such a class may require changing the class thus making it prone to instability.

Only top-level classes are reported.

Use the Maximum number of transitive dependencies field to specify the maximum allowed number of direct or indirect dependencies for a class.

Available only from Code | Inspect Code or Code | Analyze Code | Run Inspection by Name and isn't reported in the editor.
